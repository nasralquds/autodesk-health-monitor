2. Information the App Accesses Locally

To perform its diagnostic functions, the App accesses the following information locally on your computer only:

2.1 System Information

  • Available RAM and memory usage
  • Disk space availability
  • Display DPI scaling settings
  • GPU driver information and TDR settings
  • .NET Framework version
  • UAC (User Account Control) level
  • Pending Windows updates status
  • Temporary folder size
  • Virtual memory / page file settings
  • System uptime
  • User profile size

2.2 Autodesk Product Information

  • Installed Autodesk product names, versions, and installation paths
  • Autodesk application log files (read-only access for error analysis)
  • Installation file integrity verification
